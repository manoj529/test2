Ionic Todo App
=====================
A basic "todo" app built with the Ionic Framework.

Getting Started
---------------
`git clone https://github.com/ecofic/ionic-todo-app.git`

`cd ionic-todo-app`

`npm install`

`ionic serve`

Enjoy!
